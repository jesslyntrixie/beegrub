## ğŸ’¼ Business Model

- **Service Fee**: Rp 3,000 per order (paid by students)
- **Vendor Revenue**: 100% of menu price goes to vendors
- **No Commission**: Zero commission taken from vendors
- **Example**: 
  - Menu price: Rp 25,000
  - Vendor receives: Rp 25,000
  - Student pays: Rp 28,000 (menu + service fee)
